# Примеры использования анализатора с фильтрацией по дате

## Быстрый старт

### 1. Анализ только сегодняшних данных
```bash
python run_all_ultra.py --today
```
Это самый быстрый способ проанализировать данные за текущий день.

### 2. Анализ конкретной даты
```bash
python run_all_ultra.py --date 2025-11-03
```

### 3. Анализ диапазона дат
```bash
python run_all_ultra.py --start-date 2025-11-01 --end-date 2025-11-03
```

### 4. Анализ всех доступных данных (по умолчанию)
```bash
python run_all_ultra.py
```

## Расширенные примеры

### Анализ с пользовательским числом воркеров
```bash
# Анализ сегодняшних данных с 16 воркерами
python run_all_ultra.py --today --workers 16

# Анализ диапазона дат с 8 воркерами
python run_all_ultra.py --start-date 2025-11-01 --end-date 2025-11-03 --workers 8
```

### Частичный анализ
```bash
# Анализ данных с определенной даты до сегодня
python run_all_ultra.py --start-date 2025-11-02

# Анализ данных до определенной даты
python run_all_ultra.py --end-date 2025-11-02
```

## Сценарии использования

### Ежедневный мониторинг
Запускайте каждый день для анализа свежих данных:
```bash
python run_all_ultra.py --today
```

### Тестирование стратегий на исторических данных
```bash
# Анализ недельного периода
python run_all_ultra.py --start-date 2025-10-27 --end-date 2025-11-03

# Анализ конкретного дня
python run_all_ultra.py --date 2025-11-02
```

### Инкрементальные обновления
Обрабатывайте только новые данные, указав дату начала:
```bash
# Обработать данные с 2 ноября и далее
python run_all_ultra.py --start-date 2025-11-02
```

## Формат даты

Все даты должны быть в формате **YYYY-MM-DD** (год-месяц-день).

Примеры:
- ✓ Правильно: `2025-11-03`, `2025-01-15`, `2024-12-31`
- ✗ Неправильно: `03-11-2025`, `2025/11/03`, `11-03-2025`

## Преимущества фильтрации по дате

1. **Скорость**: Анализ только нужных данных значительно быстрее
2. **Фокус**: Можно сосредоточиться на интересующем периоде
3. **Экономия ресурсов**: Меньше нагрузка на память и процессор
4. **Гибкость**: Легко тестировать разные временные периоды

## Помощь

Для просмотра всех опций:
```bash
python run_all_ultra.py --help
```
