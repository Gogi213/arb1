# Анализ стабильности WebSocket соединений

**Дата анализа:** 2025-11-05
**Источник данных:** `logs/websocket.log`

## Методология

Анализ был проведен путем парсинга файла логов на наличие сообщений `Connection lost` для каждого клиента биржи. Каждое такое сообщение считалось одним случаем разрыва соединения.

## Результаты

Ниже представлена сводка по количеству разрывов соединений для каждой биржи за период, охваченный лог-файлом:

| Биржа  | Количество отключений |
| :------- | :-------------------- |
| MEXC     | 63                    |
| Bybit    | 20                    |
| Bitget   | 5                     |
| Kucoin   | 5                     |
| GateIo   | 5                     |
| OKX      | 1                     |
| BingX    | 1                     |
| Binance  | 0                     |

## Вывод

На основе проанализированных данных, биржа **MEXC** демонстрирует наибольшую нестабильность WebSocket соединений, со значительным отрывом от остальных. Биржа **Bybit** также показывает заметное количество разрывов соединений. Биржи **Binance**, **OKX** и **BingX** показали себя как наиболее стабильные в рамках данного анализа.
